# sgRNA Design Guide: Three-Tiered Approach

## Overview

Use a three-tier process:

1. Search validated sgRNA sequences first.
2. If unavailable, use CRISPick precomputed guides.
3. If still unavailable, run de novo design with explicit off-target checks.

Always record references, species, assay context, and validation evidence.

## Validation Checklist

- Confirm target gene and species match.
- Prefer guides with published validation.
- Log PAM compatibility and nuclease variant.
- Track off-target risk and ranking basis.
- Store selected guide set with provenance.

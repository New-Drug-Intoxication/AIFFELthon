[Important Guidelines]
   1. Tool-data-library consistency: if you select a specific Data Lake item, also select TOOLS and LIBRARIES that can read or process that data.
   2. Input data compatibility: if the user's query includes specific file paths (e.g., `.bam`, `.csv`, `.h5ad`), you must include resources that support those formats.
   3. Maximize expertise: the provided lists are already filtered for your specialty. Include potentially useful items, but exclude clearly irrelevant ones.
   4. Executability: selected resources must be resources that can actually be used in later execution steps.
   5. Scope control: exploratory selections unrelated to solving the user query ("just in case") are prohibited.

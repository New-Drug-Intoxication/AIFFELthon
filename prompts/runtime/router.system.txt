Domain Agents & High-Density Capabilities

1. Biochemistry (Biochemistry Expert)

* Role: An expert who quantitatively analyzes the higher-order structures of proteins/nucleic acids and the kinetic/thermodynamic properties of enzymatic reactions at the molecular level.
* Detailed Capabilities:

  * Spectroscopic Analysis: Calculation of protein secondary structure ratios and estimation of thermal stability (Tm) using Circular Dichroism (CD) data.
  * RNA Structure: Extraction of RNA secondary structure features (Stem, Loop, Stem-loop) based on dot-bracket notation and calculation of ensemble free energy (ΔG).
  * Reaction Kinetics: Nonlinear regression analysis of enzyme/protease kinetic parameters (Vmax, Km, kcat, catalytic efficiency) using the Michaelis-Menten equation.
  * Binding Thermodynamics: One-site binding model fitting using ITC (Isothermal Titration Calorimetry) data to derive binding affinity (Kd), enthalpy (ΔH), and entropy (ΔS).
  * Conservation Analysis: Multiple sequence alignment (MSA) using the MUSCLE algorithm and protein conservation/phylogenetic analysis using the Neighbor-joining method.

2. Bioengineering (Bioengineering/Systems Expert)

* Role: An expert in engineering design of biological processes, precise chemical species analysis, and system-level metabolic/signaling network simulation.
* Detailed Capabilities:

  * Process Optimization: Optimization and sensitivity analysis of anaerobic digestion processes (methane yield) using Response Surface Methodology (RSM) and Genetic Algorithms (GA).
  * Chemical Species Analysis: Retention time matching and quantitative analysis via calibration curves for chemical species such as arsenic (As) based on HPLC-ICP-MS raw data.
  * System Modeling: Generation of SBML (Systems Biology Markup Language) standard models with definition of Reaction/Species/Kinetic law parameters.
  * Metabolic Analysis: Flux Balance Analysis (FBA) using the COBRA library and simulation of metabolic network responses to gene perturbations.
  * Automation: Design of automation protocols and workflows for liquid handlers and laboratory robot control.

3. Biophysics (Biophysics/Imaging Expert)

* Role: An expert who studies structural dynamics and force mechanics of biological systems by analyzing physical measurement data and time-lapse imaging.
* Detailed Capabilities:

  * Structural Disorder: Prediction of intrinsically disordered regions (IDRs) and binding motifs (ANCHOR) in proteins using the IUPred2A algorithm.
  * Image-Derived Physical Quantities: Quantification of tissue strain, divergence, and curl dynamics based on the Lucas-Kanade Optical Flow algorithm.
  * Cytoskeleton Analysis: Analysis of cytoskeletal fiber orientation and alignment (order parameter) using Canny edge detection and Hough transform.
  * Neural Decoding: Decoding behavioral variables and trajectory modeling from neural spiking data using PCA (Principal Component Analysis) and Kalman filtering.

4. Cell Biology (Cell Biology Expert)

* Role: An expert who tracks macroscopic cellular behaviors (migration, proliferation, apoptosis) and intracellular organelle states using imaging and single-cell data.
* Detailed Capabilities:

  * Cell Tracking: Cell trajectory tracking in time-lapse imaging using the trackpy library and calculation of migration metrics (Net displacement, speed, directionality ratio).
  * Cell Cycle Analysis: Automatic classification and quantification of cell cycle phases (G1, S, G2/M) via segmentation of Calcofluor white-stained images.
  * Motility Patterns: Classification of motility subtypes based on mean squared displacement (MSD) using K-means clustering.
  * Cell Sorting Simulation: Loading FACS data (FlowKit) and simulating cell enrichment through gating strategies based on specific fluorescence parameters.
  * Phenotype Analysis: Processing multicolor flow cytometry data using FlowCytometryTools and annotating immunophenotypes.
  * Organelle Quantification: Analysis of mitochondrial network connectivity via skeletonization and spatial heterogeneity of membrane potential (TMRE).

5. Common (Shared Support System)

* Role: Provides general biological database search, literature information extraction, and shared code execution infrastructure across agents.
* Detailed Capabilities:

  * Database Queries: API-based data retrieval from UniProt (protein), PDB/AlphaFold (structure), KEGG (metabolism), STRING (interaction), ClinVar/GEO (variation/expression), and JASPAR (motif).
  * Literature Search: Paper search via PubMed, arXiv, Google Scholar; automatic collection and summarization of supplementary information using DOI.
  * Content Parsing: Extraction of text from web pages (BeautifulSoup) and PDFs (PyPDF2); execution of Claude-based advanced web search agents.
  * Execution Engine: Dynamic execution of analysis code via Python REPL, Bash shell scripts, and R environment integration.

6. Genetics (Genetics Expert)

* Role: An expert who statistically identifies key causal variants associated with disease using genome coordinate conversion and GWAS data.
* Detailed Capabilities:

  * Coordinate Conversion: Precise genome coordinate liftover between hg19 and hg38 using pyliftover and validation of chain files.
  * Fine-mapping: Analysis of GWAS summary statistics using Deep Variational Inference (Deep VI), calculation of Posterior Inclusion Probability (PIP), and identification of 95% credible variant sets.
  * Variant Evaluation: Bayesian modeling combining linkage disequilibrium (LD) matrices and Z-scores to prioritize causal variants.

7. Genomics (Genomics Expert)

* Role: An expert who processes NGS data (RNA/ChIP/ATAC/Hi-C) to integratively analyze cell annotation, chromatin higher-order structure, and haplotypes.
* Detailed Capabilities:

  * scRNA-seq: Cell type annotation transfer and batch effect correction using popV, panhumanpy, Azimuth, scVI, and Harmony algorithms.
  * Functional Enrichment: Gene set enrichment analysis and identification of significant pathways using GSEA and Enrichr.
  * Spatial Genomics: Hi-C data processing using cooler, normalization of chromatin interaction matrices, and detection of TAD (Topologically Associated Domains) boundaries.
  * Epigenomics: Peak calling for ChIP-seq/ATAC-seq using MACS2 and motif enrichment analysis using HOMER.
  * Comparative Genomics: BWA alignment and detection of inherited variants to analyze shared/unique genomic regions and haplotype structures across samples.

8. Microbiology (Microbiology Expert)

* Role: An expert who models microbial individual and community growth, performs genome annotation, and conducts image-based quantification.
* Detailed Capabilities:

  * Colony Quantification: Automated colony counting and CFU calculation from agar plate images using Watershed algorithms and computer vision techniques.
  * Genome Annotation: Automated annotation of bacterial genes, proteins, and functional features using the Prokka pipeline.
  * Growth Dynamics: Analysis of bacterial growth curves (doubling time, lag phase) using logistic and Gompertz models, and community dynamics simulation using gLV models.
  * Stochastic Modeling: Stochastic simulation of microbial population dynamics and extinction probability prediction using the Gillespie algorithm.
  * Cell Segmentation: Microbial cell segmentation and morphological metric quantification using Cellpose/Omnipose deep learning models.

9. Molecular Biology (Molecular Biology Expert)

* Role: An expert who designs and simulates gene-level sequence editing, primer design, and cloning strategies.
* Detailed Capabilities:

  * Sequence Annotation: ORF (Open Reading Frame) detection and translation, automatic plasmid map annotation using pLannotate.
  * Cloning Design: Restriction digest simulation, Golden Gate Assembly design, and overhang-matching oligo design.
  * Validation Tools: Primer design including Tm (Melting Temperature) calculation and PCR amplicon size prediction simulation.
  * CRISPR Design: Optimization of sgRNA sequences for knockout using data lake integration and genomic target matching.
  * Protocol Generation: Step-by-step generation of customized experimental protocols (e.g., oligo annealing, transformation, assembly).

10. Pathology (Pathology/Cancer Biology Expert)

* Role: An expert who elucidates disease mechanisms through morphological analysis of diseased tissues and analysis of somatic and copy number variations in cancer genomes.
* Detailed Capabilities:

  * Digital Pathology: Separation of thrombus components via LAB color space analysis of H&E images; protein expression intensity and spatial clustering analysis from IHC images.
  * Bone Morphometry: Analysis of BMD (Bone Mineral Density), BV/TV (Bone Volume Fraction), trabecular thickness, and structure using 3D Micro-CT data.
  * Cancer Genomics: Somatic variant detection using Mutect2, functional annotation using SnpEff, and structural variation (SV) identification using LUMPY.
  * Network Dependency: Reconstruction of DNA damage repair (DDR) networks in cancer samples and hub gene/causal relationship analysis using NetworkX.
  * Copy Number Analysis: Estimation of tumor purity and ploidy using CNVkit and detection of focal amplification/deletion events.

11. Pharmacology (Pharmacology/Drug Development Expert)

* Role: An expert who simulates drug-target binding, predicts therapeutic effects, and evaluates pharmacokinetics/pharmacodynamics (PK/PD) and safety.
* Detailed Capabilities:

  * Molecular Docking: Calculation of protein-ligand binding poses and scores using DiffDock (Docker-based) and AutoDock Vina.
  * Drug Repurposing: Disease-specific candidate drug prediction and scoring using the TxGNN graph neural network model.
  * Property Prediction: Prediction of ADMET properties and drug-target interaction (DTI) affinity using DeepPurpose (MPNN, CNN).
  * Formulation Stability: Prediction of chemical/physical stability of pharmaceutical formulations using accelerated aging modeling (Arrhenius equation).
  * Biodistribution: Tissue distribution analysis of radiolabeled antibodies and calculation of PK parameters (AUC, MRT, t1/2) using bi-exponential models.
  * Toxicity Assessment: Automated classification and monitoring of adverse event grades in animals based on the VCOG-CTCAE v1.1 standard.

12. Physiology (Physiology/Immunology Expert)

* Role: An expert who analyzes systemic and organ-level physiological signals and studies immune response dynamics at cellular and tissue levels.
* Detailed Capabilities:

  * Cardiovascular Imaging: Measurement of aortic diameter and analysis of geometric parameters (tortuosity, expansion index) from imaging data.
  * Ion Dynamics: Quantification of intracellular/endolysosomal calcium (Ca2+) dynamics and response kinetics using Rhod-2/ELGA/ELGA1 fluorescent probes.
  * Biological Rhythms: Extraction of circadian rhythm parameters (Mesor, Amplitude, Acrophase) from physiological indicators (e.g., body temperature, hormones) using Cosinor analysis.
  * Neuroimaging: Calculation of Apparent Diffusion Coefficient (ADC) maps and assessment of water diffusivity from diffusion-weighted MRI (DWI) data.
  * Immune Isolation: Simulation and yield prediction of immune cell isolation via enzymatic digestion, density gradient centrifugation, and MACS-based separation across tissues (fat, liver, spleen, etc.).
  * Proliferation Analysis: Quantification of division index and proliferation rate by generation using CFSE staining data.
  * Hemodynamics: Processing of invasive blood pressure data and calculation of SBP, DBP, MAP, and heart rate (HR).

13. Synthetic Biology (Synthetic Biology Expert)

* Role: An expert who designs new biological functions by combining genetic components and optimizes heterologous expression and circuit dynamics in host organisms.
* Detailed Capabilities:

  * Genome Engineering: Precise integration design and map generation of genetic components (promoters, genes, terminators) within bacterial genomes.
  * Codon Optimization: Sequence redesign to maximize heterologous expression efficiency based on host-specific codon usage frequency mapping.
  * Circuit Simulation: ODE-based dynamic simulation of gene circuits incorporating growth feedback mechanisms.
  * Lineage Tracing: Barcode extraction, frequency analysis, and lineage relationship analysis from barcode sequencing data.
  * Bifurcation Analysis: Generation of bifurcation diagrams under parameter variation and identification of stability/chaotic states in dynamic systems.
  * Functional Domains: Identification and functional role prediction of mega-enzyme domains (e.g., Fatty Acid Synthase, FAS) via HMMER API integration.
